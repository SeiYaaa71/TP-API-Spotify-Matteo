# Manuel d'utilisation — Site Web API

## Introduction
Ce programme est conçu pour faciliter la gestion et le suivi du projet API. Il offre des fonctionnalités pour organiser les tâches, suivre l'avancement et collaborer efficacement.

## Installation

1. **Cloner le dépôt :**
    ```bash
    git clone https://github.com/SeiYaaa71/TP-API-Spotify.git
    ```
2. **Accéder au dossier :**
    ```bash
    cd TP-API-Spotify
    cd site web
    ```
## Utilisation
    ``` bash
    go run .
    ```
